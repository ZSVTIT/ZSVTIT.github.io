项目说明

发表于 2019-08-01 | 分类于 项目 | 本文总访问量 次

开发背景

  大学的教材，多且贵，而且大部分的教材的使用率不高，若是每本教材都买新的，那么有点费钱又费资源。所以不少学生都选择使用二手书，我们的二手书的交易重要是二手书店，但是呢，二手书店都是低价买入我们的二手书，却是以比较高的价格卖给我们。基于以上的现象，我就打算做一个二手书交易平台，方便同学们进行卖书买书。

项目的意义

  • 二手书交易没有中间商赚差价。
  • 平台的用户都是校园内的用户，既能保证信息的有效性，又能保证交易的快速进行（不用快递嘛）。

开发环境

  • 操作系统：Windows家庭版
  • 开发工具：Intellij IDEA + Sublime Text3
  • 数据库：Mysql 5.7.26
  • Java SDK：Oracle JDK 1.8.0_211

部署环境

  • 操作系统：Linux Ubuntu Server 16.04 X64
  • 虚拟化技术：Docker
  • 服务器：腾讯云

项目管理工具

  • 项目构建：Maven + Nexus
  • 代码托管：Git+GitHub
  • 镜像管理：Docker Registry

后台主要技术

  • 核心框架：SpringBoot+SpringCloud

  • 视图框架：SpringMVC

  • ORM框架：mybatis(使用tk.mybatis来简化mybatis开发)

  • 页面引擎：Thymeleaf

  • 数据库连接池：Alibaba Druid

  • 数据库缓冲：Redis Sentinel

  • 接口文档引擎：Swagger2 RESTful 风格 API 文档生成

  • 分布式链路追踪：ZipKin

  • 分布式文件系统：Alibaba FastDFS

  • 分布式服务监控：Spring Boot Admin

  • 分布式协调系统(注册中心)：Spring Cloud Eureka

  • 分布式配置中心：Spring Cloud Config

  • 反向代理负载均衡：Nginx

前端主要技术

  • 前端框架：Bootstrap + jQuery
  • 前端模版：AdminLTE

其他工具或插件

  • 接口测试工具：Postman
  • Redis可视化工具：RedisDesktopManager
  • MySQL图像化管理工具：SqlYog
  • 终端模拟工具：Xshell
  • 分页插件：DataTables
  • 表单插件：iCheck
  • 富文本编辑器：wangEditor
  • 文件上传：Dorpzone
