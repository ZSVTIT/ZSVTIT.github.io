指令

ls系列

  • ls//显示不隐藏的文件和文件夹
  • ls -a //显示当前目录下的所以文件及文件夹（包含隐藏的.和..等）
  • ls -l //显示不隐藏的文件及文件夹的详细信息
  • ls -la (等于ls -l -a，也等于ll)

mkdir系列

  • mkdir directory //创建新的目录
  • mkdir -p fatherDirectory sonDirectory //父目录不存在的情况下，先生成父目录，也就是说可以一下生成多个目录

cd系列

  • cd directory //切换到指定的目录
  • 单独一个cd //回到当前用户的主目录(就是回到了home)

touch系列

  • touch file //生成一个新的空文件

echo系列

  • echo "String" //在屏幕上输出字符串String
  • echo “String”>>文件名.格式 //把“String”写进文件里面（新建文件的形式）

cp系列

  • cp file1 file2 //将文件file1备份为文件file2（拷贝）
  • cp file ...directory //将file等文件复制到目录directory下

cat系列

  • cat file //显示file的内容
  • cat file1 file2 //依次显示file1和file2的内容
  • cat > file //新建文件file，并且马上从键盘输入内容，按Ctrl+D表示输入完成，回车保存退出。
  • cat file1 file 2 > file3 //将file1和file2的内容合并成新的文件file3，同时file1和file2依旧存在。

find系列

  • find file//在当前目录下查找文件

rm系列

注意：通过rm删除操作，无法恢复。

  • rm file//删除文件file
  • rm -rf directory //删除目录directory，不管里面有没有文件

mv系列

  • mv file driectory //将文件file从当前目录移动到目录directory下

tree系列

注意：要先安装tree，命令为

  • 以树状图列出当前目录的内容

grep系列

  • grep String/char file //从指定的文件里查找指定的字符或者字符串，字符和字符串都不出''或”“

文件解压系列

  • zip格式：unzip file

vi/vim系列

简介

就是文本编辑器。`vim`是`vi`的升级版

vi/vim键盘图

vi/vim的使用

基本上vi/vim 共分为三种模式，分别是命令模式（Command mode），输入模式（Insert mode）和底线命令模式（Last line mode）。 这三种模式的作用分别是：

  • 命令模式

    用户刚刚启动vi/vim，便进入了命令模式。此时敲击键盘动作会被识别为命令，而非输入字符。以下是几个常用的几个命令。

    • i：切换到输入模式。
    • x：删除当前光标所在的字符。
    • :：切换到底线命令模式，就在最低一层输入命令。

  • 输入模式

    同windows的输入

  • 底线命令模式

    在命令模式下按下:(英文的冒号)就能进入底线命令模式了。

    在底线命令模式中，基本的命令有：

    • q：退出程序。
    • w：保存文件。
    • wq：保存文件并退出

vim 文件名.格式：就是用vim/vi打开了指定文件。

nano

简介

`nano`是一个字符端的文本编辑器，有点像`DOS`下的`editor`。它比` vi/vim` 要简单得多，比较适合 Linux 初学者使用。

命令

  • 保存：Ctrl + O
  • 搜索：Ctrl + w
  • 上一页：ctrl + y
  • 下一页：ctrl + v
  • 退出：ctrl + x
