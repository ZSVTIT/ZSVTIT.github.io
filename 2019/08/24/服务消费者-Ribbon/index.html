服务消费者(Ribbon)

概述

  服务消费者：服务的调用方。

  Ribbon 是一个负载均衡（ 分摊到多个操作单元上进行执行，这里的操作单元可以是服务器，这里的例子是端口号）客户端，可以很好的控制 http 和 tcp 的一些行为。

大白话

  服务消费者就是开放给用户来调用服务提供者的。

  Ribbon调用服务提供者的方法：在服务接口中

RestTemplate

RestTemplate的介绍

是Spring用于同步client端的核心类，简化了与http服务的通信，并满足RestFul原则，程序代码可以给它提供URL，并提取结果。

Service中的getForObject()是RestTemplate请求方法中的一种。

RestTemplate的请求和Http请求的对比。

Http Method RestTemplate Method
DELETE delete
GET getForObject(返回消息体)
getForEntity
HEAD headForHeaders
OPTIONS optionsForAllow
POST postForLocation
postForObject
PUT put
any exchange
execute

如果只关注，返回的消息体，可以直接使用postForObject。

getForEntity方法的返回值是一个ResponseEntity<T>，ResponseEntity<T>是Spring对HTTP请求响应的封装，包括了几个重要的元素，如响应码、contentType、contentLength、响应消息体等。

@Autowired的用法和作用

这个注解可以对类成员变量、方法及构造函数进行自动装配的工作。通过`@Autowired`的使用来消除`set`,`get`方法。用了`@Autowired`的话，只需要声明即可。
